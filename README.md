# Study Agent

面向上海市小学二、三、四、五年级学生的 AI 辅导 Agent，支持数学、英语、语文的出题和答疑。

## 功能特性

- 🎓 **年级适配**：针对 2-5 年级学生定制难度
- 📚 **三科覆盖**：数学、英语、语文
- 📝 **智能出题**：根据年级和知识点生成练习题
- 💬 **答疑解惑**：回答学生的学习问题
- 🧠 **大模型驱动**：使用 MiniMax M2.1 模型

## 快速开始

```bash
npm install
npm run dev
```

## 项目结构

```
study-agent/
├── src/
│   ├── index.js      # 入口文件
│   ├── agent.js      # Agent 核心逻辑
│   └── subjects/     # 学科模块
│       ├── math.js
│       ├── english.js
│       └── chinese.js
├── knowledge/        # 知识库
├── tests/            # 测试文件
└── prompts/          # 对话模板
```

## 使用方法

与 Agent 对话：
- "帮我出 5 道数学计算题"
- "解释一下分数是什么"
- "我想做一篇英语阅读理解"

## 目标用户

上海市小学 2-5 年级学生及家长
