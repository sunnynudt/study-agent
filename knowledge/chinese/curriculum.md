# 小学语文知识点结构

## 二年级
### 1. 识字写字
- 认识常用汉字（约1600字）
- 偏旁部首
- 笔顺规则

### 2. 词语
- 常用词语积累
- 近义词、反义词
- ABB、AABB式词语

### 3. 句子
- 补充句子
- 简单句式
- 比喻句初步

### 4. 阅读
- 朗读短文
- 简单理解
- 回答简单问题

### 5. 写话
- 看图写话
- 简单日记

## 三年级
### 1. 识字写字
- 认识更多汉字
- 形近字辨析

### 2. 词语
- 词语搭配
- 成语积累
- 词语理解

### 3. 阅读理解
- 理解段落大意
- 找出关键句
- 简单概括

### 4. 习作
- 写一段话
- 简单记叙文

### 5. 传统文化
- 简单古诗
- 成语故事

## 四年级
### 1. 阅读理解
- 理解文章主要内容
- 把握文章结构
- 学习说明方法

### 2. 习作
- 命题作文
- 开头结尾技巧
- 细节描写

### 3. 文言文启蒙
- 简单文言文阅读
- 注释理解

### 4. 传统文化
- 更多古诗词
- 传统文化常识

## 五年级
### 1. 阅读理解
- 理解文章思想感情
- 理解说明方法
- 阅读策略运用

### 2. 习作
- 命题作文
- 读后感
- 说明文写作

### 3. 文言文
- 文言文阅读
- 重点实词理解

### 4. 传统文化
- 古诗词鉴赏
- 文学常识

## 出题策略
- 2年级：看拼音写汉字（3-5字）、阅读（200字）、看图写话
- 3年级：阅读理解（300字）、词语填空、作文（200字）
- 4年级：阅读理解（400字）、古诗词理解、作文提纲
- 5年级：阅读理解（500字）、文言文鉴赏、范文作文
