# 小学英语知识点结构

## 二年级
### 1. 字母
- 26个字母大小写
- 字母发音基础

### 2. 基础词汇
- 颜色类：red, blue, yellow, green...
- 数字类：1-20
- 动物类：cat, dog, bird, fish...
- 水果类：apple, banana, orange...
- 家庭类：father, mother, brother...

### 3. 简单句型
- This is... / That is...
- I am... / You are...
- I have... / You have...
- What's this? / What's that?

### 4. 日常对话
- 问候与告别
- 介绍自己
- 简单问路

## 三年级
### 1. 核心词汇
- 衣物类：shirt, dress, pants, shoes...
- 身体部位：head, face, hand, foot...
- 动作词：run, jump, sing, dance...
- 时间词：morning, afternoon, evening

### 2. 语法基础
- 单数/复数
- 代词：I, you, he, she, it, we, they
- 动词第三人称单数

### 3. 句型
- Can you...? / Yes, I can. / No, I can't.
- I like... / I don't like...
- What do you like?

### 4. 阅读
- 100词左右短文
- 简单的故事

## 四年级
### 1. 词汇扩展
- 交通工具：bus, taxi, bike, train...
- 场所：school, hospital, supermarket...
- 方位词：near, far, in front of, behind

### 2. 语法
- 现在进行时
- 一般现在时
- 简单疑问句

### 3. 阅读理解
- 150-200词短文
- 理解大意
- 细节查找

### 4. 写作
- 写句子
- 写简单段落

## 五年级
### 1. 核心词汇
- 更多日常词汇
- 形容词比较级
- 动词过去式基础

### 2. 语法
- 一般过去时
- 一般将来时（will）
- 简单复合句

### 3. 阅读理解
- 200-300词文章
- 主旨大意
- 细节理解
- 推理判断

### 4. 写作
- 看图写话
- 命题作文（80词左右）
- 书信基础

## 出题策略
- 2年级：看图写词、连线、简单对话
- 3年级：选词填空、阅读理解（100词）
- 4年级：阅读理解（200词）、中译英
- 5年级：阅读理解（300词）、作文
